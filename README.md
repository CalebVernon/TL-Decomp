# TL-Decomp

## Overview

TL-Decomp is an early-stage reverse engineering project focused on Tomodachi Life for the Nintendo 3DS. The goal is to analyze and document core game systems, such as islander data structures, friendship values, and event logic, in order to provide a foundation for future work and potential forks.

This project is intended for educational purposes and to facilitate collaborative research into game mechanics.

## Goals

1. Identify key data structures (friendship, love events, islander stats)  
2. Map relevant functions and memory addresses  
3. Document findings clearly to allow others to build upon them  
4. Maintain a clean, accessible codebase suitable for contributors of all skill levels
